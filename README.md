
# Azure AI Search - Busca Cognitiva

Seguindo o guia Explore an Azure AI Search index (UI) -> https://microsoftlearning.github.io/mslearn-ai-fundamentals/Instructions/Labs/11-ai-search.html

## 🚀 Como Configurar
1. Criar um serviço de AI Search no Azure.
2. Criar um índice e definir seus campos.
3. Indexar dados utilizando API ou conectores.
4. Realizar consultas de busca.

## 🔍 Exemplos de Aplicação
- Busca de produtos em e-commerces.
- Recuperação inteligente de documentos.
- Pesquisa semântica com IA.
